# Api tools
Anything potentially useful from calculator to tempreature converted or UUIDv7 generatoer - should be here

> [!IMPORTANT]
> This tool is WIP (Work in progress) and may contain bugs and be incomplete.

What's available:

| Endpoint        | Description                                                     |
| --------------- | --------------------------------------------------------------- |
| /id/uuid        | pseudorandom UUIDv4                                             |
| /id/uuidv7      | Random UUIDv7                                                   |
| /crypto/sha256  | Hasher                                                          |
| /network/subnet | Subnet related calculations with ip address and subnet provided |

Most of the endpoints also allow a Get argument `format` to optionally return only in `text` format instead of json
